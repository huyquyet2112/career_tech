# CareerTech

Website hỗ trợ tuyển dụng và tìm kiếm việc làm.

## 🚀 Tính năng chính

- Đăng ký & đăng nhập cho ứng viên và nhà tuyển dụng
- Tìm kiếm và ứng tuyển việc làm
- Quản lý hồ sơ ứng viên
- Quản lý tin tuyển dụng
- Phân quyền (RBAC)
- Dashboard thống kê

## 🛠️ Công nghệ sử dụng

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- jQuery & Bootstrap 4
- DataTables
- Font Awesome
